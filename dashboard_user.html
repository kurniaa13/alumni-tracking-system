ℹ️ Tentang Sistem

Alumni Tracking System

Versi: 1.0

Deskripsi: Sistem informasi untuk tracking dan pendataan alumni institusi pendidikan secara terpusat dan terintegrasi.

Fitur Utama

  • 📝 Pendataan alumni dengan informasi lengkap
  • 💼 Pencatatan informasi pekerjaan dan posisi
  • 📱 Integrasi sosial media (LinkedIn, Instagram, Facebook, TikTok)
  • 🔐 Sistem keamanan berbasis login
  • 📊 Dashboard analitik untuk administrator
  • 📖 Direktori alumni yang dapat diakses pengguna

Keamanan Data

Semua data alumni disimpan secara lokal di browser Anda. Data tidak dikirim ke server manapun. Dilarang menggunakan data alumni untuk kepentingan diluar pembelajaran dan penelitian institusi.

Fitur untuk Alumni

  • Kelola profil pribadi dan data pekerjaan
  • Kelola akun sosial media perusahaan
  • Lihat direktori alumni lainnya
  • Cari alumni berdasarkan nama atau status pekerjaan
